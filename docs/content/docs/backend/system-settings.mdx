# 系统配置数据结构

系统配置保存在 `settings` 表中，目前只使用两行：

| key | 说明 |
| --- | --- |
| `public` | 公开配置，前端可以读取 |
| `private` | 私有配置，只给后端和管理员使用 |

## public.value

```json
{
  "modelChannel": {
    "availableModels": ["默认渠道::gpt-5.5", "图片渠道::gpt-image-2"],
    "textModels": ["默认渠道::gpt-5.5"],
    "imageModels": ["图片渠道::gpt-image-2"],
    "videoModels": [],
    "audioModels": [],
    "modelInfos": [
      { "value": "图片渠道::gpt-image-2", "provider": "图片渠道", "model": "gpt-image-2", "displayName": "GPT Image 2", "type": "image", "maxSize": "2k" }
    ],
    "modelCosts": [
      { "model": "默认渠道::gpt-5.5", "credits": 1 },
      { "model": "图片渠道::gpt-image-2", "credits": 10 }
    ],
    "defaultModel": "默认渠道::gpt-5.5",
    "defaultImageModel": "图片渠道::gpt-image-2",
    "defaultVideoModel": "",
    "defaultTextModel": "默认渠道::gpt-5.5",
    "defaultAudioModel": "",
    "systemPrompt": "",
    "allowCustomChannel": true,
    "modelTypeRules": {
      "textModels": "gpt-*",
      "imageModels": "gpt-image-*",
      "videoModels": "seedance-*",
      "audioModels": "tts-*"
    }
  },
  "auth": {
    "allowRegister": true,
    "linuxDo": {
      "enabled": false
    }
  }
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `modelChannel` | object | 模型渠道公开配置组 |
| `auth` | object | 认证相关公开配置 |

`modelChannel` 字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `availableModels` | string[] | 系统可用模型选项值；后台渠道模型会按 `服务商::模型名` 暴露，避免不同服务商的同名模型被合并 |
| `textModels` | string[] | 可用于文本能力的模型选项值列表 |
| `imageModels` | string[] | 可用于图片能力的模型选项值列表 |
| `videoModels` | string[] | 可用于视频能力的模型选项值列表 |
| `audioModels` | string[] | 可用于音频能力的模型选项值列表 |
| `modelInfos` | object[] | 前端可读取的模型展示元信息，来自后台模型清单 |
| `modelCosts` | object[] | 模型算力点配置，后端模型接口调用前按模型选项值预扣，上游失败时返还；未配置默认不扣除 |
| `defaultModel` | string | 默认模型，从 `availableModels` 中选择；为空或失效时优先选择文本模型 |
| `defaultImageModel` | string | 默认图片模型，从 `imageModels` 中选择 |
| `defaultVideoModel` | string | 默认视频模型，从 `videoModels` 中选择 |
| `defaultTextModel` | string | 默认文本模型，从 `textModels` 中选择 |
| `defaultAudioModel` | string | 默认音频模型，从 `audioModels` 中选择 |
| `systemPrompt` | string | 系统提示词 |
| `allowCustomChannel` | boolean | 是否允许用户在配置弹窗中切换为本地直连渠道，默认允许 |
| `modelTypeRules` | object | 模型类型匹配规则，每行一个 glob 或 `/regex/` |

`modelInfos` 每项字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `value` | string | 前端选择器使用的模型选项值，通常是 `服务商::模型名` |
| `provider` | string | 服务商或渠道名称 |
| `model` | string | 真实模型名称；后端转发上游接口时使用该值 |
| `displayName` | string | 前端展示名称 |
| `type` | string | 模型类型，可用逗号保存多个类型 |
| `maxSize` | string | 图片模型最大清晰度限制：`auto`、`1k`、`2k`、`4k`，非图片模型可为空 |

`modelCosts` 每项字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `model` | string | 模型选项值，后台渠道通常为 `服务商::模型名` |
| `credits` | number | 每次后端模型接口调用前预扣的算力点 |

用户侧请求模式：

| 模式 | 说明 |
| --- | --- |
| 云端渠道 | 使用后端 `/api/v1/*` 代理接口，请求会按模型选项值解析服务商和真实模型名，再匹配 `private.value.channels` 中的可用渠道 |
| 本地直连 | 默认可选；`allowCustomChannel` 关闭后不可选，用户在浏览器本地配置 `baseUrl`、`apiKey` 和模型列表后直接请求模型接口 |

`auth` 字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `allowRegister` | boolean | 是否允许用户注册，默认允许；关闭后注册入口隐藏，注册接口拒绝新用户创建 |
| `linuxDo.enabled` | boolean | 是否开启 Linux.do 登录 |

## private.value

```json
{
  "channels": [
    {
      "protocol": "openai",
      "name": "默认渠道",
      "baseUrl": "https://api.example.com",
      "apiKey": "sk-xxx",
      "models": ["gpt-5.5", "gpt-image-2"],
      "type": "",
      "weight": 1,
      "enabled": true,
      "remark": ""
    }
  ],
  "promptSync": {
    "enabled": true,
    "cron": "*/5 * * * *"
  }
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `channels` | object[] | 模型渠道列表 |
| `promptSync` | object | GitHub 远程提示词定时同步配置 |

`channels` 每项字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `protocol` | string | 协议，当前为 `openai` |
| `name` | string | 渠道名称 |
| `baseUrl` | string | OpenAI 兼容接口地址 |
| `apiKey` | string | 渠道密钥 |
| `models` | string[] | 该渠道可用模型 |
| `type` | string | 渠道模型默认类型：`text`、`image`、`video`、`audio`，空值表示按规则自动识别 |
| `weight` | number | 渠道权重；同一模型有多个可用渠道时按权重随机 |
| `enabled` | boolean | 是否启用 |
| `remark` | string | 备注 |

后端调用模型时，会先检查后台模型清单中该模型是否启用，再从已启用、已配置 `baseUrl` 和 `apiKey`、且 `models` 包含目标模型的渠道中选择一个。

`promptSync` 字段：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `enabled` | boolean | 是否开启定时同步，默认开启 |
| `cron` | string | Cron 表达式，默认每 5 分钟 |
